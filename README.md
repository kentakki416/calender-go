# Requirement
 
* react 18.2.0
* redux 4.2.0
* redux-thunk 2.4.1
* hogehuga 1.0.2
* dayjs 1.11.3
* Material-UI

# Usage
 
DEMOの実行方法など、"hoge"の基本的な使い方を説明する

```bash
git clone https://github.com/hoge/~
cd examples
python demo.py
```
